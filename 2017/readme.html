VS Drop Assist

Usage

Drag one or more objects from the Solution Explorer and drop onto a code window.

Drop whilst holding down the following buttons to generate different text. After dropping, the %v% is selected and you can immediately type a new name (press ESC to cancel the selection)


If dragging 1-or-more class members (use ctrl or shift-click to select multiple items in the solution explorer)
KeyString
ALT %v%.%m%.Value = ;\n
SHIFTColumns.Add(%v%.%m%);\n
(none) %v%.%m%,\n

If dragging at least 1 class (use ctrl or shift-click to select multiple items in the solution explorer)
KeyString
SHIFT var %v% = new %f%();\n
(If dropping into a method)
SHIFT public readonly %f% %v% = new %f%();\n
(If dropping outside a method within a class declaration)
(none) %f%\n

Tokens:

TokenReplaced with
%f%Class full name
%v%Class type name
%mMember (method) name
